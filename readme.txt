== Description ==
This plugin allows you to check phone numbers against TCPA litigators and Do Not Call (DNC) complainants.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/tcpa-litigator-checker/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > TCPA Checker** to configure API credentials.

== Frequently Asked Questions ==
= How does the API authentication work? =
You need an API key from [TCPA Litigator List](https://tcpalitigatorlist.com/) to access the phone checking service.
